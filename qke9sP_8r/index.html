校对随笔 - 20190610

今天想和大家分享的是，在使用塔多思的过程中，我所遇到的一个问题以及问题的解决方法。

今天早上在我准备使用塔多思的时候，我突然发现无法打开需要校对的文件。每当双击这些文件的时候，就会弹出以下这些内容：

Not able to edit files > The specified item is already part of a command bar

问题有些棘手。毕竟昨天我关机的时候一切都很正常。

虽然我能看懂每一个单词的意思，也能看懂整句话的意思，但是我不知道应该怎么做。

鉴于今天之前所有的软件都可以正常运行，那么我首先就怀疑问题出现在软件的缓存。通过腾讯桌面自带的文件搜查功能，我很快就定位到了塔多思软件的缓存位置（一般是c盘的user name名下的一些文件夹）。

之后将缓存全部清除，同时关闭塔多思，重启电脑。

好了，新的一天工作又可以开始了。
